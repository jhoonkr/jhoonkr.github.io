#네이버 고객센터

서비스명
네이버 고객센터
진행기간
2020.05 (1달 간)
주요내용
모바일 페이지 마크업
본인이 기여한 점
  • 재사용성, 랜더링 속도를 고려한 컴포넌트 UI 구조화 작업
  • 모바일 페이지 사용성을 고려한 UI 스타일 속성 구현 (버튼 클릭 영역 등)
사용한 기술스택 및 지식
HTML5, CSS, CSS3, Jquery 플러그인 활용
결과 및 성과

중요도 있는 업무로, 일정이 촉박한 업무였습니다. 작업 시작 전 컴포넌트 형태의 UI 구현을 통해 재사용성, 렌더링 속도 개선에 힘을 주어 개발을 진행하였습니다. 성공적으로 배포 후 빠른 시간 내에 업무 대응을 잘 해주었다는 평가를 받았습니다. 적절한 일정 조율과 빠른 대응의 좋은 사례가 되었던 업무입니다.

  • 팀 내에서 App Project 진행 과정에 관한 내용 정리 및 공유
  • 페이지 컴포넌트화
